# JavaScript Tasks

This repository contains simple JavaScript projects for learning DOM manipulation and event handling.

## Projects

### 1. Multi Color
- Changes the background to different predefined colors.
- Demonstrates JavaScript event handling.

### 2. Random Color
- Generates a random background color on button click.
- Uses JavaScript Math functions.

### 3. Theme Switcher
- Switches between different themes (Light/Dark).
- Built using HTML, CSS, and JavaScript.

## Technologies Used
- HTML5
- CSS
- JavaScript 

## How to Run

   ```bash
   git clone https://github.com/your-username/Javascript-Task.git
   ```
